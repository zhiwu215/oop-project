# 后端技术选型

| 技术        | 说明               |
| ----------- | ------------------ |
| SpringBoot  | Web应用开发框架    |
| MybatisPlus | ORM框架            |
| MySQL       | 数据库             |
| Session     | 跨请求存储用户信息 |

# 功能

## **用户注册和登录**

- 使用md5对用户密码进行加密存储
- 在登录时，使用threadlocal+session进行用户信息存储和身份区分
